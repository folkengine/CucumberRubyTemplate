CucumberRubyTemplate
====================

Baseline cucumber project for my explorations with running Ruby Cucumber inside IntelliJ 12

Requirements
------------

* Ruby 1.9.x installed and on your classpath:

```sh
    $> ruby -v
    ruby 1.9.3p362 (2012-12-25) [i386-mingw32]
```    
* IntelliJIDEA 12
